Installation

Run the following commands to install and configure your miner.

💡 Copy and paste these commands into your terminal. The wizard will detect your platform automatically.

install.sh 
# Download and install RustChain Miner
curl -fsSL https://rustchain.network/install.sh | bash

# Configure your miner
rustchain-miner config --wallet YOUR_WALLET --pool pool.rustchain.network:3333 --worker worker1
