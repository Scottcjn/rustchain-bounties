# RustChain Miner Setup Wizard

A beautiful, interactive setup wizard for configuring RustChain mining nodes.

## Features

- **5-Step Guided Setup**: Platform selection → Wallet config → Mining settings → Installation → Success
- **Auto-Platform Detection**: Automatically detects Windows, macOS, or Linux
- **Wallet Generation**: Built-in wallet address generator for new users
- **Multi-Platform Support**: Windows, macOS, Linux, and Docker
- **Mobile-Friendly**: Responsive design works on all devices
- **Copy-to-Clipboard**: One-click copying of installation commands
- **No Backend Required**: Pure HTML/CSS/JS - runs entirely in the browser

## Usage

Simply open `index.html` in any modern web browser. No server required.
