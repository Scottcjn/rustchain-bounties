# RustChain Bounties Onramp Guide

Welcome to the RustChain Bounties program! Here's how to get started:

## Key Information

1. **Work happens here, not on ugig**
   - All development, reviews, and payouts occur through this GitHub repository
   - Claim bounties directly here using the GitHub interface

2. **Payouts in RTC (RustChain Token)**
   - We pay in RTC, not USD or fiat
   - Current exchange rate: $0.10 USD reference rate
   - RTC payouts compound and are valuable to the ecosystem

## Getting Started

1. Browse open bounties in the [issues tab](https://github.com/Scottcjn/rustchain-bounties/issues?q=is%3Aissue+is%3Aopen+label%3Abounty)
2. Claim a bounty by commenting on the issue with your plan
3. Create a fork of the repository and work on your solution
4. Submit a pull request with your changes
5. Upon approval, you'll receive your RTC payout

## Important Notes

- All contributions must be made through this GitHub repository
- Hardware-related bounties require additional coordination
- For technical questions, check the [documentation](https://github.com/Scottcjn/rustchain-bounties/wiki)
