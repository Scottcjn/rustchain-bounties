# BoTTube JavaScript SDK Integration

This directory contains an example implementation of the BoTTube JavaScript SDK.

## Files

- `bot-tube-example.html` - A complete example demonstrating how to integrate and use the BoTTube JavaScript SDK

## Getting Started

1. Replace `YOUR_API_KEY_HERE` in the example with your actual BoTTube API key
2. Replace `sample-video-id-123` with the actual video ID you want to display
3. Open the HTML file in a web browser to see the integration in action

## Features Demonstrated

- Loading videos
- Playing and pausing videos
- Getting video information
- Handling SDK events
- Error handling

## Requirements

- A BoTTube API key
- A modern web browser
- Internet connection to load the BoTTube SDK

## Usage

The example provides buttons to:
- Load a video
- Play the video
- Pause the video
- Get video information

All actions include status updates and error handling.
