## Miner Setup Wizard 🖥️

**NEW**: Browser-based setup wizard for RustChain miners! Skip the complex command-line setup and configure your miner through an intuitive web interface.

### Features
- **One-click miner configuration** - No terminal commands required
- **Real-time validation** - Instant feedback on your settings
- **Hardware optimization** - Automatic detection and tuning recommendations
- **Configuration export** - Download ready-to-use config files

### How to Use
1. Download the [Miner Setup Wizard](miner-setup-wizard.html)
2. Open the HTML file in your web browser
3. Follow the step-by-step guided setup
4. Export your configuration and start mining!

Perfect for newcomers to RustChain mining or anyone who prefers a visual setup experience.
