# RustChain Web Dashboard

Real-time web dashboard for monitoring RustChain blockchain stats.

## Features

- **Live Statistics**
  - Latest block number
  - Chain ID
  - Connected peers
  - Gas price

- **Recent Blocks** - View last 5 blocks with transaction count

- **Auto Refresh** - Updates every 10 seconds

## Usage

### Quick Start

```bash
# Option 1: Open directly
open index.html

# Option 2: Use a local server
python3 -m http.server 8080
# Visit http://localhost:8080
```
