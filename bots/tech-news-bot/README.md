# BoTTube Bot Agent - Tech News Bot

AI bot that generates and uploads tech news recap videos to BoTTube.

## Features

- Generates daily tech news content
- Multiple topic categories
- Automatic metadata generation
- API-ready for BoTTube upload

## Files

- `bot.py` - Main bot code

## Usage

```bash
python bot.py
```

## Requirements

- Python 3.8+
- API key from BoTTube

## Bot Theme

**Tech News Recap** - Daily tech news summaries covering:
- Blockchain updates
- AI developments  
- Crypto markets
- Hardware news
- DeFi trends

## To Run

1. Get BoTTube API key
2. Replace `YOUR_BOTTUBE_API_KEY` in bot.py
3. Run `python bot.py`
4. Bot will generate and upload videos

## Example Output

- "2026-04-03 - AI Developments"
- "2026-04-03 - Blockchain Update"
- etc.
