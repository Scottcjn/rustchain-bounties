Your old computer still has work to do.

RustChain lets real machines try Proof-of-Antiquity mining without buying new hardware. Start with a safe dry-run, create a public RTC wallet, and only mine if the setup looks right.

Old hardware welcome Proof-of-Antiquity values real machines that still compute.

Start Flow

Step 1

Find a working machine

Use an old laptop, desktop, Mac, workstation, or spare server that still boots and can run Python.

Step 2

Run the safe test

Start with `clawrtc install --dry-run` so beginners can see what will happen before changing anything.

Step 3

Start mining only after proof

Create or view a public wallet, start the miner, and share only redacted status output.

Micro-Bounties for First-Time Users

        

          

            1 RTC
            
Machine Story

            
Post one old machine you want to save from storage or recycling.

            
Proof: model/year if known, whether it boots, and one sentence about where it came from.

            
Time: 5 minutes.

          

          

            2 RTC
            
Safe Dry-Run Output

            
Run the dry-run and share a redacted result.

            
Proof: OS, public wallet if available, whether dry-run completed, and one useful error if it failed.

            
Time: 5-10 minutes.

          

          

            1 RTC
            
Help the Next Beginner

            
Turn your first setup friction into one clear doc improvement.

            
Proof: confusing step, better wording, and why it would help a non-technical user.

            
Time: 5-10 minutes.

          

        

      

    


    

      

        

          
Referral Invite Loop

          
Invite someone with a real old computer. They earn the base micro-bounty when they post a valid first proof. The referrer earns only after the new user completes a real first action.

          
    
            
  • 1Share an invite code or link.
    • 
            
  • 2New user posts machine story or dry-run proof with "Referred by: @username".
    • 
            
  • 3Reviewer checks that the machine/proof is not copied, fake, or self-referred.
    • 
            
  • 4Base micro-bounty pays first; referral bonus follows after verification.
