# 60-Second Video Script — RustChain Origin Story
## "The Machine That Still Works"

---

**[VISUAL: Close shot of a dusty laptop being pulled from a drawer. PowerBook G4. Beat-up but intact.]**

**NARRATOR (V.O.):**
This machine still works.

**[VISUAL: Screen lights up. Fan hums. Desktop loads.]**

**NARRATOR (V.O.):**
Apple stopped supporting it in 2007.
No app will run on it.
No cloud service will take it.
The recycling center won't touch it.

**[VISUAL: Screen shows a rejection screen from a modern download — "This app requires macOS 11 or later."]**

**NARRATOR (V.O.):**
Every system we've built says: throw it away.

**[VISUAL: Hand hovers over trash can. Pause. Doesn't drop it.]**

**NARRATOR (V.O.):**
I couldn't do it.

**[VISUAL: Terminal window opens. A command is typed.]**

**NARRATOR (V.O.):**
Then I found RustChain.

**[VISUAL: Text on screen: "Proof-of-Antiquity. Older hardware earns more."]**

**NARRATOR (V.O.):**
A blockchain where old hardware earns *higher* rewards.
Not despite its age.
*Because* of it.

**[VISUAL: Block explorer showing miner attestation. A wallet ID appears in the live feed.]**

**NARRATOR (V.O.):**
This PowerBook — 2003, PowerPC processor —
earns a 2.5x multiplier.
The hardware proves itself.
The network pays it.

**[VISUAL: Drawer opens again. A Raspberry Pi 2. A ThinkPad. An old Linux box. Each gets picked up.]**

**NARRATOR (V.O.):**
I kept going.

**[VISUAL: Split screen: four terminal windows, four machines mining.]**

**NARRATOR (V.O.):**
Eleven miners across four nodes on two continents.
Small network. Real network.
You can see your machine show up the moment it connects.

**[VISUAL: GitHub page — github.com/Scottcjn/RustChain]**

**NARRATOR (V.O.):**
If you have old hardware that still boots —
try it.
Five minutes. One command.
The drawer doesn't have to be a graveyard.

**[VISUAL: PowerBook screen. Miner running. Block confirmed. Clean fade to logo.]**

**TEXT ON SCREEN:**
RustChain
github.com/Scottcjn/RustChain

**NARRATOR (V.O.):**
If it still computes, it still has value.

## Production Notes

- **Total runtime**: ~58 seconds at measured narration pace
- **Tone**: Quiet, personal, unhurried — no hype, no music swell, no dramatic cuts
- **Visual language**: Natural light, real hardware, no motion graphics
- **Key emotional beat**: The pause before the machine gets dropped in the trash — the decision not to throw it away
- **Call to action**: GitHub URL, held on screen for 4+ seconds
- **Do NOT include**: Price predictions, follower counts, VC endorsements, claims about "thousands of miners"
