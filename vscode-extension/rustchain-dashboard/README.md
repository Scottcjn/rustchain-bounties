# RustChain Dashboard VS Code Extension

VS Code extension showing RustChain wallet balance, miner status, and bounty board.

## Features

- 💰 Wallet Balance in status bar
- ⛏️ Miner Status indicator  
- 📋 Bounty Browser sidebar
- ⏱️ Epoch Timer countdown

## Installation

1. Clone this repo
2. Run `npm install`
3. Run `npm run compile`
4. Press F5 to debug

## Usage

1. Open VS Code settings
2. Set `rustchain.wallet` to your wallet name
3. Status bar shows RTC balance
4. Sidebar shows bounties and miner status

## Requirements

- VS Code 1.75+
- Node.js 18+
