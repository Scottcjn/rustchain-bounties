# RustChain Dashboard - VS Code Extension

Shows RustChain wallet balance, miner status, bounty board, and epoch timer in VS Code sidebar.

## Features

1. **Wallet Balance** — RTC balance in status bar + sidebar
2. **Miner Status** — Green/red indicator for miner attestation
3. **Bounty Browser** — Browse open bounties from rustchain-bounties repo
4. **Epoch Timer** — Countdown to next epoch settlement
5. **Quick Actions** — Claim bounty, open node, copy wallet address

## Setup

1. Install: 
2. Set wallet: Settings → RustChain → walletAddress
3. Set node URL (optional): Settings → RustChain → nodeUrl
