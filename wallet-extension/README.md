# RustChain Wallet Browser Extension - 40-100 RTC Bounty

## Goal
Chrome/Firefox extension with MetaMask Snap support

## Features
- RTC wallet management
- Send/Receive transactions
- Balance checking
- MetaMask Snap integration
- Transaction history

## Files
- `manifest.json` - Extension manifest
- `popup.html` - UI
- `background.js` - Background service
- `snap/` - MetaMask Snap code
