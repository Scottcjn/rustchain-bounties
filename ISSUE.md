## Bounty: Add Your Name to CONTRIBUTORS.md ˇŞ 5 RTC

**Difficulty:** Beginner (no coding required)
**Reward:** 5 RTC
**Time:** ~15 minutes

This bounty teaches you the full GitHub contribution workflow ˇŞ fork, edit, pull request. Once you've done this, you'll know how to contribute to any open source project.

---

### Requirements

- A GitHub account (free at github.com)
- Be a member of the RustChain Discord server
- That's it!

---

### Step-by-Step Instructions

**Step 1: Fork the repository**
1. Go to https://github.com/Scottcjn/Rustchain
2. Click the **"Fork"** button in the top-right corner
3. This creates your own copy of the repo under your GitHub account

**Step 2: Navigate to CONTRIBUTORS.md**
1. In YOUR forked repo (it will say `YourUsername/Rustchain` at the top)
2. Click on the file `CONTRIBUTORS.md`
3. Click the **pencil icon** (Edit this file) in the top-right of the file view

**Step 3: Add your entry**
1. Scroll to the bottom of the file
2. Add a new line following this format:

```
| @YourGitHubUsername | YourDiscordHandle#0000 | What you're interested in (e.g., mining, development, testing, community) |
```

For example:
```
| @coolguy42 | CoolGuy#1234 | Interested in mining on vintage hardware |
```

**Step 4: Commit your changes**
1. Scroll down to "Commit changes"
2. In the commit message, write: `Add @YourUsername to CONTRIBUTORS.md`
3. Select **"Create a new branch for this commit and start a pull request"**
4. Click **"Propose changes"**

**Step 5: Create the Pull Request**
1. You'll be taken to the "Open a pull request" page
2. Title: `Add @YourUsername to CONTRIBUTORS.md`
3. In the description, write: `Claiming bounty from issue #XX` (replace XX with this issue number)
4. Click **"Create pull request"**

**Step 6: Claim your bounty**
1. Come back to THIS issue and leave a comment with a link to your PR
2. Once merged, 5 RTC will be sent to your wallet

### FAQ

**Q: What if CONTRIBUTORS.md doesn't exist yet?**
A: Create it! That's even more impressive. Use the format above with a header row.

**Q: I messed up my fork, what do I do?**
A: Delete your fork and re-fork. No harm done.

**Q: Can multiple people claim this?**
A: Yes! Every person who adds themselves gets 5 RTC. This bounty stays open.
