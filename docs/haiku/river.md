# Haiku: River

Five RTC per unique haiku collection.

## Flow

```
Water moves on
Endless journey to the sea
Current carries life
```

## Current

```
Swift or gentle
River shapes the land below
Erosion creates
```

## Bridge

```
Spans the waters
Connection over the flow
Paths intersect
```

## Fish

```
Under the surface
Life teems in cold waters
Swimmers thrive
```

## Bank

```
Sedges hold roots
Shoreline supports ecosystems
Green borders blue
```

## Delta

```
River meets sea
Sediment builds new land
Birth of islands
```

## Peace

```
Sound of water
Calming rhythm never stops
Meditation flows
```

## Source

```
Mountain spring starts
Single drop becomes a stream
Beginning of journey
```
