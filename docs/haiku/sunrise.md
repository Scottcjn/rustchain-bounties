# Haiku: Sunrise

Five RTC per unique haiku collection.

## Morning

```
Darkness retreats
First light appears on horizon
Day is born
```

## Colors

```
Orange and pink
Paint the sky in soft hues
Morning glory
```

## Hope

```
New day arrives
Opportunities await us
Promise of sun
```

## Birds

```
Songbirds awaken
Welcome the returning light
Chorus begins
```

## Early

```
World still quiet
Streets empty, peaceful morning
Solitude treasures
```

## Energy

```
Warmth returns
Light energizes all things
Life continues
```

## Coffee

```
First sip awaits
Sunrise and caffeine together
Perfect morning
```

## Routine

```
Daily ritual
Sun rises, we rise too
Consistent cycle
```
