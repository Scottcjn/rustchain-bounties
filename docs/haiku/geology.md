# Haiku: Geology

Five RTC per unique haiku collection.

## Rocks

```
Stone holds the ages
Layers of ancient stories
Earth remembers all
```

## Volcanoes

```
Fire from below
Mountain awakens with rage
Lava writes new land
```

## Earthquakes

```
Ground shivers hard
Tectonic plates rearrange
Nature's power shown
```

## Fossils

```
Ancient life preserved
Stone captures millennium
Past comes alive
```

## Crystals

```
Hidden underground
Beauty forms in darkness
Minerals shine bright
```

## Mountains

```
Peaks touch the sky
Wind carves the ancient stone
Summit sees far
```

## Oceans

```
Floor unexplored
Depths hold secrets untold
Life thrives below
```

## Time

```
Millions of years
Rocks tell the planet's story
Patience in stone
```
