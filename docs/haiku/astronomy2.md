# Astronomy2 Haiku Collection

## About
This collection explores the science of stars and planets through haiku poetry. From observation to exploration, celebrate the cosmos.

## Haiku Categories

### Celestial Bodies
- stars and constellations
- planets and moons
- asteroids and comets
- galaxies and nebulae

### Observation Tools
- telescopes
- binoculars
- cameras
- mounts

### Space Missions
- past missions
- current probes
- future exploration
- Mars rovers

## Sample Haiku

```
Stars light the night
Ancient photons reach my eye
Looking back in time
```

```
Telescope points
Distant worlds come into view
Cosmic wonder grows
```

```
Rocket launches
Humanity reaches outward
Next horizon calls
```
