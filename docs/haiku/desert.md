# Haiku: Desert

Five RTC per unique haiku collection.

## Sand

```
Golden grains spread
Endless dunes shift with the wind
Desert breathes slowly
```

## Heat

```
Scorching sun beats
Dry air cracks the earth
Life adapts or leaves
```

## Oasis

```
Green in the brown
Water draws all living things
Miracle in sand
```

## Cactus

```
Spines protect
Stored water inside survives
Patience rewarded
```

## Camel

```
Ship of the desert
Endures the barren lands
Built for survival
```

## Dunes

```
Sculpted by wind
Patterns dance on the sand
Art of nature
```

## Night Sky

```
Stars brighter here
No city lights to compete
Galaxy reveals
```

## Survival

```
Life thrives hard
Creatures find clever ways
Desert teaches resilience
```
