# Haiku: Poetry

Five RTC per unique haiku collection.

## Words

```
Rhymes dance in lines
Emotions flow on paper
Poets speak truth
```

## Inspiration

```
Muse visits silent
Ideas bloom in quiet mind
Verse is born free
```

## Metaphor

```
Comparing reveals
Lion's heart, rose's bloom
Truth through imagery
```

## Emotion

```
Feelings expressed
Joy, sorrow, love, and hope
Words heal the soul
```

## Haiku

```
Five seven five beats
Nature in fewest words
Moment captured
```

## Sonnet

```
Fourteen lines sing
Love sonnets never die
Shakespeare lives on
```

## Free Verse

```
No rules to follow
Thoughts flow like gentle stream
Freedom in words
```

## Reader

```
Poem is complete
Only when read by another
Words come alive
```
