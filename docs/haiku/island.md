# Haiku: Island

Five RTC per unique haiku collection.

## Paradise

```
Palm trees sway slow
White sand meets turquoise sea
Remote sanctuary
```

## Isolation

```
World feels far away
Only ocean in between
Peace found in distance
```

## Native

```
Unique species found
Isolated evolution
Rare treasures live
```

## Tropical

```
Warm weather year
Sunset paints the water gold
Ever summer feels
```

## Exploration

```
Coastlines to find
Hidden coves, secret beaches
Adventures await
```

## Community

```
Small population
Everyone knows each other
Close-knit islanders
```

## Surrounded

```
Water all around
Isolation is complete
Freedom in boundaries
```

## Sunset

```
Day ends in gold
Island silhouette against sky
Beauty overwhelms
```
