# Mars Haiku Collection

## About
This collection celebrates Mars and the red planet through haiku poetry. From rovers to future colonies, explore our neighbor.

- colonies

## Sample Haiku

```
Red dust kicks up
Rover explores alone
Mars examined
```

```
Olympus rises
Largest volcano in solar system
Giant waits
```

```
Future colonists
Build homes beneath the surface
Humanity spreads
```
