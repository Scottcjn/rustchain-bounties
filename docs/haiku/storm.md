# Haiku: Storm

Five RTC per unique haiku collection.

## Thunder

```
Rumble in sky
Nature's drums announce arrival
Power unleashed
```

## Lightning

```
Flash cracks the dark
Branches illuminate briefly
Electric display
```

## Rain

```
Drops percussion
On rooftops, leaves, and streets
Storm symphony
```

## Wind

```
Gusts howl and push
Trees bend under the force
Nature reshapes
```

## Dark

```
Skies turn to black
Day becomes midnight briefly
Drama in clouds
```

## After

```
Rainbow appears
Sun returns after the tempest
Beauty follows
```

## Power

```
Nature's strength shown
Electricity in motion
Respect the storm
```

## Safe

```
Shelter welcomed
Protected from the elements
Peace in home
```
