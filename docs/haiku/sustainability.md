# Haiku: Sustainability

Five RTC per unique haiku collection.

## Earth

```
Our only home
Planet spins in cosmic sea
Treasure to protect
```

## Green Energy

```
Sun and wind power
Clean energy without end
Future is bright
```

## Recycling

```
Waste becomes new
Circle of life continues
Nothing is lost
```

## Conservation

```
Protect the wild
Forests, oceans, creatures all
Nature must live
```

## Climate

```

Small steps lead to big impact
Together we can
```

## Water

```

Every sip matters
```

## Soil

```
Earth feeds us all
Healthy dirt grows healthy food
Nurture the ground
```

Tomorrow's world
Built by today's choices
Leave it better
