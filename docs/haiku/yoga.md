# Haiku: Yoga

Five RTC per unique haiku collection.

## Breath

```
Inhale, exhale
Life force flows through the body
Balance restored
```

## Pose

```
Mountain stands tall
Warrior prepares for battle
Peace in each posture
```

## Stretch

```
Muscles awaken
Flexibility returns slow
Body remembers
```

## Meditation

```
Mind becomes still
Thoughts like clouds drift away
Awareness shines
```

## Flexibility

```
Bend without break
 supple like the willow
Strength in yielding
```

## Peace

```
Inner calm flows
Stress dissolves in each pose
Serenity returns
```

## Energy

```
Chakras align
Vital force moves freely
Spirit awakened
```

## Community

```
Mats together
Breathing in unified rhythm
Souls connected
```
