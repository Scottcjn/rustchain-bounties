# Haiku: Gardening

Five RTC per unique haiku collection.

## Soil

```
Rich earth receives
Seeds that hold future life
Garden begins
```

## Growth

```
Sun and water help
Green shoots reach for the sky
Nature provides
```

## Flowers

```
Blooms bring beauty
Colors paint the landscape
Fragrance fills air
```

## Harvest

```
Fruits of labor
Vegetables ready to pick
Plenty to share
```

## Patience

```
Time required
Gardening cannot be rushed
Growth takes season
```

## Seasons

```
Spring, summer, fall
Each brings different tasks
Nature's rhythm
```

## Tools

```
Trowel and rake
Essential helping hands
Work made easier
```

## Peace

```
Therapeutic calm
Hands in soil, mind at ease
Garden heals soul
```
