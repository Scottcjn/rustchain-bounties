# Haiku: Meditation

Five RTC per unique haiku collection.

## Silence

```
Quiet the mind
Stillness within meditation
Peace found within
```

## Focus

```
Breath is anchor
Present moment is all there is
Awareness grows
```

## Inner Peace

```
Chaos outside fades
Still waters run deep within
Calm embraces all
```

## Mindfulness

```
Here and now matters
Each moment lived completely
Life becomes rich
```

## Stillness

```
Motionless body
Quiet mind, listening within
Deep tranquility
```

## Letting Go

```
Release attachments
Things come and go like waves
Be the ocean
```

## Compassion

```
Loving kindness
Extends to self and others
All beings matter
```

## Awakening

```
Spirit rises
Illumination comes slowly
Truth illuminates
```
