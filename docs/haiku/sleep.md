# Haiku: Sleep

Five RTC per unique haiku collection.

## Rest

```
Eyes close at last
Body recovers its strength
Dreams await soon
```

## Dream

```
Imaginations
Stories play in sleeping mind
Mystery unfolds
```

## Night

```
Darkness embraces
Stars witness the quiet world
Peace in sleeping
```

## Pillow

```
Soft landing head
Comfort in the darkness
Sanctuary found
```

## Insomnia

```
Sleep won't arrive
Counting sheep in endless night
Tired but awake
```

## Alarm

```
Morning arrives
Rude awakening pulls us back
New day begins
```

## Snore

```
Sound fills the room
Others struggle to rest
Noisy nights pass
```

## Refreshed

```
Waking refreshed
Energy returns to body
Ready for the day
```
