# Haiku: Winter

Five RTC per unique haiku collection.

## Cold

```
Bitter winds blow
Frost paints the windows white
Winter is here
```

## Snow

```
White blankets earth
Snowmen rise from frozen flakes
Children delight
```

## Fireplace

```
Warmth in the home
Crackling wood brings comfort
Cozy evenings
```

## Holidays

```
Festive season
Family gatherings happen
Joy multiplies
```

## Short Days

```
Darkness early
Sun sets before we expect
Lights help us cope
```

## Hot Chocolate

```
Warm sweet drink
Marshmallows float on top
Comfort in cup
```

## Hibernation

```
Animals sleep
Bears and more rest for months
Nature pauses
```

## Reflection

```
Year ends soon
Time for contemplation
New year ahead
```
