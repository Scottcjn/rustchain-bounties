# Haiku: Summer

Five RTC per unique haiku collection.

## Heat

```
Sun blazes hot
Long days of warmth and sunshine
Summer arrives
```

## Beach

```
Sun and sand call
Vacation time begins now
Leisure in sun
```

## Vacation

```
Time away from work
Family adventures await
Memories made
```

## Ice Cream

```
Cold sweet treat
Beating the summer heat
Creamy delight
```

## BBQ

```
Grilling outside
Smoke and delicious aromas
Flavor develops
```

## Pool

```
Cool water refreshes
Splashing brings the relief
Summer essential
```

## Thunderstorm

```
Afternoon rain
Relief from the hot day
Drama in sky
```

## Late

```
Evening extends
Fireflies dance in the warm air
Magic in night
```
