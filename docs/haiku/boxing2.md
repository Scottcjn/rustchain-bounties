# Boxing2 Haiku Collection

## About
This collection celebrates boxing and the sweet science through haiku poetry. From training to the fight, explore the combat sport.

## Haiku Categories

### Techniques
- jabs and crosses
- hooks and uppercuts
- footwork
- defense

### Training
- road work
- bag work
- sparring
- conditioning

### The Fight
- weigh-ins
- ring entrances
- rounds
- decision or knockout

## Sample Haiku

```
Gloves wrap tight
Training builds the fighter
Strength meets skill
```

```
Jab snaps out fast
Counter comes right back
Dance of the ring
```

```
Knockout lands
Opponent hits the canvas
Champion rises
```
