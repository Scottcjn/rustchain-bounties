# Haiku: Writing

Five RTC per unique haiku collection.

## Words

```
Letters form thoughts
Sentences build the world
Author creates
```

## Story

```
Beginning sparks
Middle builds, ending satisfies
Narrative complete
```

## Character

```
People in ink
Living through the author's pen
Fiction becomes real
```

## Setting

```
Places come alive
Worlds built within pages
Imagination wins
```

## Revision

```
First draft is raw
Polishing brings out the gem
Perfect emerges
```

## Dialogue

```
Voices on page
Characters speak their truth
Conversations breathe
```

## Emotion

```
Reader feels too
Joy, sorrow, love, and fear
Words touch the soul
```

## Publishing

```
Manuscript flies
From desk to bookstore shelves
Dreams become real
```
