# Vinyl Haiku Collection

## About
This collection celebrates vinyl records and analog music through haiku poetry. From classic albums to crate digging, explore the warm sound of vinyl.

## Sample Haiku

```
Needle drops down
Music fills the room with warmth
Sound fills the air
```

```
Album spins round
Crackle adds character
Nostalgia plays
```

```
Crate full of finds
Treasure waiting to be found
Rare pressing discovered
```
