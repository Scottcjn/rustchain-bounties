# Haiku: Autumn

Five RTC per unique haiku collection.

## Leaves

```
Colors transform
Green becomes gold and crimson
Nature's painting
```

## Harvest

```
Abundance gathered
Fruits of the season's labor
Storage for winter
```

## Cool

```
Chill in the air
Jackets become necessary
Cozy returns
```

## Wind

```
Breezes increase
Leaves scatter across the ground
Sweeping necessary
```

## Migration

```
Birds fly south
Seasons marked by their journeys
Instinct guides
```

## Pumpkin

```
Orange everywhere
Spice and everything nice
Autumn arrives
```

## Cozy

```
Warm drinks in hand
Knitting and comfortable sweaters
 comfort
```

## Ending

```
Year grows older
Nature prepares for rest
Cycles continue
```
