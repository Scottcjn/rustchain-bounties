# Haiku: Night

Five RTC per unique haiku collection.

## Darkness

```
Stars appear bright
Night wraps the world in shadow
Quiet descends
```

## Moonlight

```
Silver illumination
Guides the nocturnal wanderer
Peace in the dark
```

## Stars

```
Constellations shine
Distant suns in the velvet sky
Infinite wonder
```

## Dreams

```
Imagination runs
Nighttime brings adventures
Reality pauses
```

## Peace

```
World becomes still
City lights dim slowly
Serenity returns
```

## Sleep

```
Bodies rest now
Tomorrow waits in darkness
Rejuvenation
```

## City

```
Streetlights guide paths
Neon signs advertise life
Night city breathes
```

## Reflection

```
Thoughts in quiet
Contemplation after dark
Wisdom emerges
```
