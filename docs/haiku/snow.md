# Haiku: Snow

Five RTC per unique haiku collection.

## Winter

```
White blanket spreads
Silent night transforms the world
Cold beauty arrives
```

## Flakes

```
Each unique form
No two snowflakes the same
Designs in ice
```

## Cold

```
Chill in the air
Breath becomes visible mist
Warmth is precious
```

## Silence

```
Sound dampened soft
World becomes quiet and still
Peace in white
```

## Ski

```
Down the slopes glide
White powder sprays behind
Adrenaline flows
```

## Build

```
Snowman takes shape
Children sculpt winter joy
Imagination creates
```

## Melt

```
Spring arrives slow
Snow drips and returns to water
Cycle continues
```

## Beauty

```
Winter painting
Monochrome with blue shadows
Elegance in white
```
