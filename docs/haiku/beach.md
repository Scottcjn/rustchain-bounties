# Haiku: Beach

Five RTC per unique haiku collection.

## Sand

```
Grains beneath feet
Warm from the sun's embrace
Beach welcomes all
```

## Waves

```
Crashing to shore
Rhythm of the ocean heart
Constant comfort
```

## Sun

```
Yellow orb sets
Painting the sky in orange
Golden hour
```

## Shell

```
Treasures washed up
Spiral shapes from the ocean
Collections grow
```

## Swim

```
Cool water calls
Diving in, refreshing escape
Summer ritual
```

## Relax

```
Lounging all day
Book and umbrella nearby
Leisure in sun
```

## Children

```
Building sandcastles
Creating kingdoms in minutes
Imagination
```

## Sunset

```
Day ends in blaze
Silhouettes on golden sea
Peaceful ending
```
