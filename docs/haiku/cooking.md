# Haiku: Cooking

Five RTC per unique haiku collection.

## Ingredients

```
Fresh from the earth
Colors and flavors combine
Culinary art
```

## Chef

```
Master creates
Flavors dance upon the plate
Magic in kitchen
```

## Taste

```
Savor each bite
Sweet, salty, sour, bitter
Flavor symphony
```

## Heat

```
Flames transform raw
Chemical changes happen
Heat creates life
```

## Family

```
Gather around
Shared meals build strong bonds
Love is on table
```

## Seasoning

```
Salt and spice add
Depth to every dish
Balance brings joy
```

## Knife Skills

```
Sharp blade moves fast
Precise cuts create beauty
Practice makes master
```

## Presentation

```
Eyes eat first too
Beautiful plating matters
Art on the plate
```
