# Haiku: Mountain

Five RTC per unique haiku collection.

## Peaks

```
Stone reaches sky
Summits touch the heavens
Glory in height
```

## Hike

```
Trail winds upward
Each step brings new vistas
Altitude calls
```

## Ridge

```
Sharply defined
Between two worlds of valley
Balance on edge
```

## Snow

```
White covers peaks
Glaciers hold ancient ice
Eternal winter
```

## Forest

```
Pine trees stand tall
Mountain hide ecosystems
Life thrives in height
```

## Summit

```
Top of the world
Reached by effort and skill
Victory achieved
```

## Trail

```
Winding paths mark
Journeys of the adventurous
Walkers leave tracks
```

## View

```
Panorama spreads
Vast landscapes below expand
Beauty overwhelming
```
