# Haiku: Rainforest

Five RTC per unique haiku collection.

## Lush

```
Green everywhere
Life multiplies abundantly
Nature's treasure
```

## Biodiversity

```
Species countless
Insects, birds, mammals, plants
Earth's pharmacy
```

## Canopy

```
Layers of green
Diverse levels of life
Jungle skyscraper
```

## Exotic

```
Strange creatures live
Colors vivid and striking
Beauty in wild
```

## Humidity

```
Moisture heavy
Rain daily refreshes
Tropical breath
```

## River

```
Waterways flow
Highways through the green maze
Life follows water
```

## Indigenous

```
Tribes live within
Ancient knowledge preserved
Culture survives
```

## Conservation

```
Protect these lands
Lungs of our planet Earth
Future needs green
```
