# Haiku: Coffee

Five RTC per unique haiku collection.

## Morning

```
First sip of day
Aromas wake the sleeping mind
Energy returns
```

## Beans

```
Dark roasted magic
Traveled from distant mountains
Cup holds the world
```

## Espresso

```
Pressure makes it strong
Rich crema tops the small cup
Pure intensity
```

## Latte

```
Cream meets the shot
Silky smooth in ceramic
Comfort in a cup
```

## Caffeine

```
Spark in every sip
Mental fog lifts away
Alertness returns
```

## Barista

```
Artisan at work
Latte art tells a story
Skill in each pour
```

## Aroma

```
Whiff of heaven
Roast fills the morning air
Noses follow
```

## Ritual

```
Daily habit
Moments of peace in chaos
Coffee time sacred
```

## Brewing

```
Pour over method
Slow drip extraction
Flavor
```

## French Press

```
Full immersion
Coarse grounds steep
Rich
```

## Cold Brew

```
Time extracts
Cold water slow
Smooth
```

## Coffee Shop

```
Third place exists
Community gathers
Space
```

## Coffee Culture

```
Global tradition
Every country brews
Unity
```

## Coffee Farming

```
Cherries ripen
Farmers harvest
Trade
```

## Coffee Roasting

```
Heat transforms
Green beans turn brown
Aroma
```

## Decaf

```
Without the buzz
Coffee flavor remains
Choice
```

## Coffee Varieties

```
Arabica sweet
Robusta strong
Blend
```
