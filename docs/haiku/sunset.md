# Haiku: Sunset

Five RTC per unique haiku collection.

## Evening

```
Day draws to close
Orange spreads across the sky
Beauty in ending
```

## Colors

```
Purple, red, gold
Sunset paints in all colors
Daily masterpiece
```

## Peace

```
Calm descends now
World slows as light fades soft
Tranquility
```

## Reflection

```
Water mirrors
The sunset becomes double
Beauty multiplies
```

## Silhouette

```
Figures stand out
Dark against the glowing sky
Simple power
```

## Memory

```
Golden moments
Captured in the mind's eye
Sunset lingers
```

## Goodbye

```
Day says farewell
Each sunset is a goodbye
And a promise
```

## Twilight

```
Between the worlds
Day and night meet in the middle
Liminal space
```
