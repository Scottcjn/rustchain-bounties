# Haiku: Sailing

Five RTC per unique haiku collection.

## Wind

```
Breezes fill sails
Boat glides across the water
Freedom on the sea
```

## Ocean

```
Endless blue expanse
Waves rock the vessel gently
Depths hold secrets
```

## Compass

```
Guiding direction
North points to destination
Stars navigate
```

## Captain

```
Master of the ship
Commands the crew with wisdom
Experience guides
```

## Horizon

```
Line where sky meets sea
Distant lands call out softly
Journey continues
```

## Anchors

```
Secure in harbor
Waiting for next adventure
Rest before voyage
```

## Storms

```
Waves crash around
Brave sailors face the weather
Courage prevails
```

## Sunset

```
Orange paints the sky
Day ends in golden beauty
Peace on the water
```
