# Haiku: Photography

Five RTC per unique haiku collection.

## Light

```
Dawn paints the sky
Captured in a single frame
Time stands still here
```

## Moment

```
Click freezes life
Instants become eternal
Memories preserved
```

## Portrait

```
Eyes tell stories
Soul captured in pixels
Truth in face
```

## Landscape

```
Nature in glory
Mountains, valleys, endless sky
Beauty framed forever
```

## Street

```
Life rushes by
Urban poetry unfolds
Stories in motion
```

## Macro

```
Tiny worlds revealed
Insects, flowers in detail
Universe in small
```

## Black and White

```
Shadows and light
Emotion without color
Timeless elegance
```

## Memory

```
Photos hold years
Faces we no longer see
Nostalgia captured
```
