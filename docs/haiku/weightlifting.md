# Weightlifting Haiku Collection

## Sample Haiku

```
Iron calls to me
Reps and sets in rhythm
Strength is built slow
```

```
Bar bends with weight
Muscles burn and grow
Tomorrow stronger
```

```
Gym floor echoes
Heavy weights collide and rise
Body transforms
```
