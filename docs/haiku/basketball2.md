# Basketball Haiku Collection

## About
This collection celebrates basketball and the game on the court through haiku poetry. From streetball to the NBA, embrace the fast-paced sport.

## Sample Haiku

```
Ball in my hands
Drive to the hoop for a dunk
Crowd goes crazy now
```

```
Three point shot
Arc is high and comes down swish
Nothing but net
```

```
Buzzer beats
One shot to win the game
Ball finds the net
```
