# Dragonfly Haiku Collection

## About
This collection celebrates dragonflies and these ancient insects through haiku poetry. From pond to garden, appreciate the delicate flyer.

## Haiku Categories

### Appearance
- iridescent wings
- long body
- compound eyes
- vibrant colors

### Behavior
- hovering
- darting
- hunting
- mating

### Habitat
- ponds
- lakes
- streams
- gardens

## Sample Haiku

```
Dragonfly rests
Wings catch the morning light
Jewels of air
```

```
Speeds over water
Catches mosquitoes for food
Eternal flyer
```

```
Eggs laid in pond
Nymphs grow beneath the surface
Transformation comes
```
