# Haiku: Morning

Five RTC per unique haiku collection.

## Dawn

```
Sun rises slow
First light paints the horizon
New day begins
```

## Sunrise

```
Golden colors spread
Night retreats from the daylight
World awakens now
```

## Coffee

```
Aromas fill home
Morning ritual begins
Energy returns
```

## Routine

```
Habits take hold
Consistency builds strength
Morning sets the tone
```

## Exercise

```
Bodies in motion
Stretch and strengthen each morning
Health is priority
```

## News

```
Headlines arrive
World updates each morning
Awareness grows
```

## Gratitude

```
Thankful for days
Each morning brings opportunity
Appreciation flows
```

## Goals

```
Daydream begins
Intentions for the day set
Dreams take action
```
