# Haiku: Spring

Five RTC per unique haiku collection.

## Bloom

```
Flowers awaken
Petals open to the sun
Color returns
```

## Rain

```
April showers help
May flowers bloom in glory
Life renews itself
```

## Nest

```
Birds build homes
New families are beginning
Generation rises
```

## Green

```
World turns verdant
Grass carpet returns to earth
Nature awakens
```

## Growth

```
Seeds become plants
Everything grows in springtime
Potential explodes
```

## Fresh

```
Clean and new feeling
Renewal in the air
Fresh starts await
```

## Allergy

```
Sneezing begins
Pollen fills the atmosphere
Nature's price paid
```

## Hope

```
Promise returns
After winter's death, life returns
Optimism grows
```
