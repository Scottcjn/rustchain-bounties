# Constellation Haiku Collection

## About
This collection celebrates constellations and star patterns through haiku poetry. From zodiac to circumpolar, map the night sky.

## Sample Haiku

```
Seven stars shine
Big Dipper in the north
Guide for travelers
```

```
Orion hunts
Belt of three bright stars
Hunter in sky
```

```
Zodiac circle
Ancient patterns in stars
Myths told for years
```
