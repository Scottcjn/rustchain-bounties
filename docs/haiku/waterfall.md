# Haiku: Waterfall

Five RTC per unique haiku collection.

## Cascade

```
Water falls down
From cliff to pool, dancing white
Nature's fountain
```

## Mist

```
Spray rises up
Rainbow forms in the mist
Magical scene
```

## Power

```
Tons of water
Unstoppable force of nature
Awe-inspiring
```

## Sound

```
Constant roar fills
White noise of rushing water
Meditation fills
```

## Pool

```
Basin receives
All the water falling down
Reflection forms
```

## Height

```
Drop is vertical
From top to bottom in seconds
Thrill of descent
```

## Beauty

```
Scenic wonder
Photos cannot capture all
Must witness in person
```

## Fresh

```
Clean air surrounds
Negative ions refresh
Wellness in nature
```
