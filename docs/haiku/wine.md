# Haiku: Wine

Five RTC per unique haiku collection.

## Vineyard

```
Grapes ripen slow
Sun warms the hanging fruit
Harvest awaits
```

## Vintage

```
Year in each bottle
Time transforms grape to gold
Patience in glass
```

## Red

```
Deep ruby flows
Tannins tell their story
Complexity sings
```

## White

```
Golden in glass
Crisp and bright, elegant
Chardonnay whispers
```

## Sparkling

```
Bubbles rise up
Celebration in every sip
Joy overflows
```

## Pairing

```
Food meets the wine
Perfect match creates magic
Senses awaken
```

## Sommelier

```
Expert guide leads
Knowledge unlocks the vintage
Appreciation grows
```

## Toast

```
Glass raised up high
Celebration begins now
Moments made golden
```
