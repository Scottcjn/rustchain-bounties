# Haiku: Transportation

Five RTC per unique haiku collection.

## Roads

```
Endless highways stretch
Vehicles chase the horizon
Journey never ends
```

## Trains

```
Steel on steel sings
Tracks connect distant cities
Travel in style
```

## Aviation

```
Wings slice the clouds
Metal birds dominate sky
Distance shrinks fast
```

## Ships

```
Oceans no barrier
Vessels float on salt water
Trade crosses seas
```

## Electric

```
Silent and clean
Future drives without fumes
Green wheels turn
```

## Bicycles

```
Human power moves
Simple machines take us far
Freedom in two wheels
```

## Public Transit

```
Together we ride
Shared journeys, shared destination
City flows as one
```

## Space Travel

```
Beyond atmosphere
Humanity reaches for stars
Earth is home no more
```
