# Haiku: Camping

Five RTC per unique haiku collection.

## Tent

```
Canvas shelter
Home for nights under stars
Simple living
```

## Fire

```
Wood crackles low
Warmth against the dark night
Stories shared
```

## Stars

```
No city lights
Millions of stars visible
Cosmic connection
```

## Nature

```
Wake in the wild
Birdsong replaces alarm clocks
Simplicity returns
```

## Hike

```
Trails to explore
Backpack carries essentials
Freedom in walking
```

## Campfire

```
Cooking over flame
Flames dance, providing warmth
Community forms
```

## Night

```
Darkness complete
Only firelight glows in tents
Quiet descends
```

## Morning

```
Sunrise wakes gently
Dew on the grass glistens
New day begins
```
