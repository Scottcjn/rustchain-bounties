## Sample Haiku

```
Ocean waves call
Paddle out through the break
Ride the rolling swell
```

```
Board cuts through water
Salt spray on my face
One with the ocean
```

```
Sunset on the beach
Waiting for the next wave
Peace in the lineup
```
