# Habits Haiku Collection

## About
This collection explores habits and daily routines through haiku poetry. From building good habits to breaking bad ones, celebrate the power of consistency.

## Haiku Categories

### Habit Types

- evening rituals
- health habits
- learning habits

### Building Habits
- starting new habits
- consistency
- habit stacking
- tracking progress

### Breaking Habits
- identifying triggers
- replacement habits
- willpower
- environmental design

## Sample Haiku

```
Morning coffee flows
Routine starts the day right
Habits shape our lives
```

```
Same action repeated
Creates pathways in the mind

```

```
Bad habit breaks
One choice at a time
Freedom begins
```
