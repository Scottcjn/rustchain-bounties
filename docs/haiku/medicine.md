# Haiku: Medicine

Five RTC per unique haiku collection.

## Healing

```
Hands that mend lives
Science meets compassion
Hope cures ill
```

## Doctors

```
White coats move fast
Saving lives is their mission
Angels in plain sight
```

## Surgery

```
Precision cuts
Steel meets flesh with care
Life hangs in balance
```

## Pharmacy

```
Pills small but mighty
Chemistry fights disease
Cure in capsule
```

## Emergency

```
Sirens in night
Racing against the clock
Every second counts
```

## Research

```
Lab lights burn late
Scientists chase the cure
Breakthrough awaits
```

## Nursing

```
Gentle care given
Tireless hands, compassionate hearts
Healing touch of love
```

## Prevention

```
Health is wealth
An ounce of prevention
Better than cure
```
