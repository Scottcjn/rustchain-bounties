# Upstream Contributions Showcase
> Last updated: 2026-04-11
## Overview
The Upstream Contributions Showcase feature was built to display elyanlabs.ai's contributions to major open-source projects, enhancing visibility and credibility within the developer community.
## How It Works
The feature is implemented in `upstream-contributions.html`, which organizes contributions into a visually appealing format using cards. Each card includes project logos, star counts, and links to the respective merged pull requests, with a clear distinction between merged and in-review contributions.
## Configuration
No configuration required.
## Usage
Simply navigate to the Upstream Contributions Showcase page on elyanlabs.ai to view the contributions.
