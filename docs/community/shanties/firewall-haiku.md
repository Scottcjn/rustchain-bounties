# Firewall Haiku

## 防火墙 俳句

### Version 1
> Signals travel fast
> Through cables made of light
> Data finds its home

### Version 2
> Bytes upon bytes flow
> 防火墙 stores them all
> Digital archive

### Version 3
> Connection secured
> 防火墙 protects the network
> Data stays safe and sound
