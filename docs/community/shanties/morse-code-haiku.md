# Morse Code Haiku

## 莫尔斯电码 俳句

### Version 1
> Zeroes and ones flow
> Connecting far and wide
> Voices meet at last

### Version 2
> Signals cross the waves
> 莫尔斯电码 carries words
> Distance fades away

### Version 3
> Through wires or airwaves
> Messages find their way home
> Technology bridges
