# Contributing to Rustchain Bounties

Welcome! This repository manages community-funded RTC bounties including the 5,000 Stars campaign (#478).

## Claiming RTC for Stars (Issue #478)
1. Create a RustChain wallet string: `your-wallet-name` (3–30 alphanumeric, `_` or `-`).
2. Star the main repo: https://github.com/Scottcjn/Rustchain (required).
3. Optionally, star other ecosystem repos for multipliers.
4. Go to Issue #478 and comment using the template from [stars.html](./stars.html):
   ```markdown
   Wallet: your-wallet-name
   https://github.com/owner/repo1
   https://github.com/owner/repo2
   ```
5. Proof is verified automatically via GitHub API; you do not need screenshots.
6. Rewards: 2 RTC for main only; 3× per repo if 6+ total; 5× per repo if all repos starred.
7. Payouts are processed by maintainers and posted in `bounties/STARS_CAMPAIGN_PAYOUTS.json`.
