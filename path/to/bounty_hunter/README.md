# Bounty Hunter
================

A tool to help you review PRs and submit review comments.

## Usage

1. Clone the repository: `git clone https://github.com/Scottcjn/rustchain-bounties.git`
2. Install the dependencies: `pip install -r requirements.txt`
3. Run the bounty hunter: `python main.py`

## Features

* Reviews PRs and provides substantive review comments
* Submits review comments to the PR
* Tracks review comments on the PR
