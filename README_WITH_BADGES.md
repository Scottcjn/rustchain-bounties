## What is RustChain?

RustChain is a revolutionary Proof-of-Age blockchain that values hardware longevity over raw processing power.

## Key Features

- **Proof-of-Age Consensus**: Rewards based on hardware age
- **Vintage Hardware Support**: PowerPC, ARMv7, and other legacy architectures
- **RTC Token Rewards**: Earn tokens for contributing to network security
- **4 Published PyPI Packages**: `clawrtc`, `beacon-skill`, `bottube`, `grazer-skill`
- **Production Network**: 15+ active miners on real hardware (PowerPC G4, G5, IBM POWER8)
- **Academic Submission**: Grail-V paper submitted to CVPR 2026 Workshop
- **Live Tokens**: wRTC on Solana (Raydium) + Base L2 (Aerodrome)
