# RustChain MCP Tool

MCP (Model Context Protocol) tool for RustChain blockchain integration.

## Features

- Get RTC balance
- Get block information
- Get transaction details
- Get network statistics

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python mcp_tool.py
```

## Tools

1. `get_balance` - Get RTC balance for an address
2. `get_block_info` - Get block information
3. `get_transaction` - Get transaction details
4. `get_network_stats` - Get network statistics
