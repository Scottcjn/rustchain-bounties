# Bounty Ledger

This ledger tracks all bounty claims and payments for the RustChain Bounties program.

## Format

Each entry records:
- **Date**: Date of the claim session
- **Wallet**: Claimant's wallet address
- **Round**: Session round number
- **Type**: PR Review, Emoji Reaction, or other
- **Amount**: RTC amount claimed
- **Status**: Pending / Approved / Paid
- **PRs/Issues**: Links to relevant PRs or issues
