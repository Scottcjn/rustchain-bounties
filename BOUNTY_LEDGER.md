# Bounty Ledger

This ledger tracks all bounty claims and payouts for the RustChain project.

## Format

Each entry includes:
- Date
- Claimant wallet address
- Number of PRs reviewed
- RTC amount
- Transaction signature (if paid)

- Payments are made weekly on Fridays
- All transactions are recorded on the Solana blockchain
