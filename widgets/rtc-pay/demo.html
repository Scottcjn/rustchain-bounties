Quick Start

Add the payment widget to your site in seconds:

<!-- 1. Include the script -->
<script src="rustchain-pay.js"></script>

<!-- 2. Add a payment button -->
<div id="rtc-pay"
     data-to="RTCyourwalletaddress..."
     data-amount="10"
     data-memo="Order #123"></div>

Programmatic Usage

Open the payment modal via JavaScript:

RustChainPay.open({
  to: 'RTCyourwalletaddress...',
  amount: '25',
  memo: 'Invoice #456',
  onSuccess: (result) => {
    console.log('Payment complete!', result);
  },
  onError: (err) => {
    console.error('Payment failed:', err);
  }
});

Configuration Options

Attribute Description Required
data-to Recipient wallet address (RTC...) Required
data-amount Payment amount in RTC Required
data-memo Optional memo/note for the payment Optional
data-node Custom RustChain node URL Optional
data-callback Webhook URL for payment notifications Optional
