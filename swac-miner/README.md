# SWAC Miner - 极简 SHA256 实现

## SWAC 架构概述

**Standards Western Automatic Computer (SWAC)** - 1950 年 NIST 建造的早期计算机

### 硬件规格
- **内存**: 256 字 × 37 位 (Williams-Kilburn 阴极射线管)
- **指令长度**: 37 位
- **寻址**: 5 位地址 (2^5 = 32 直接寻址，需要间接寻址扩展)
- **累加器**: 37 位
- **时钟频率**: ~800 kHz

### 内存限制挑战
- 256 字 = 约 1.2KB 总内存
- 需要存储：程序 + 数据 + 哈希状态
- SHA256 标准实现需要 >> 256 字

## 极简移植方案

### 策略
1. **SHA256 子集**: 仅实现核心压缩函数，预处理在外部完成
2. **流式处理**: 每次处理一个 512 位块，状态存储在累加器
3. **微控制器桥接**: 外部 MCU 负责 I/O 和网络，SWAC 仅计算

### 内存布局 (256 字)
```
地址      用途                大小 (字)
0-31      引导程序/监控        32
32-63     SHA256 常量 (K 表)   32
64-95     哈希状态 (H0-H7)     8
96-127    工作变量 (a-h)       8
128-191   消息调度 (W 表)      64
192-223   临时存储/栈          32
224-255   I/O 缓冲区           32
```
