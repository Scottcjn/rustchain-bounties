# Miner Status Notification System

Alert system for RustChain miners.

## Features

- **Discord Webhook**: Post alerts when miner misses epochs
- **Email Alerts**: Send email notifications
- **Multiple miners**: Monitor multiple wallets

## Files

- `notifier.py` - Main notification system

## Setup

1. Configure Discord webhook URL
2. Set email credentials
3. Register miner wallets

## Usage

```bash
python notifier.py
```
