# Bounties de RustChain
¡Bienvenido al programa de recompensas (bounties) de RustChain!

Este repositorio contiene las tareas abiertas para el ecosistema RustChain. Los contribuyentes pueden ganar RTC (RustChain Coins) completando tareas técnicas, de documentación o de comunidad.

## Cómo empezar
1. Encuentra un problema (issue) con la etiqueta **bounty**.
2. Haz un fork del repositorio.
3. Envía un Pull Request con tu solución.

## Reglas
- Los pagos se procesan en RTC.
- La calidad del código es revisada por el equipo de Proof-of-Antiquity (Prueba de Antigüedad).
- Proporciona tu dirección de billetera en la descripción del PR.
