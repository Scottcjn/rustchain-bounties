# RustChain Miner Installation and Running Tutorial

**Duration:** ~3 minutes

**Target Audience:** Beginners interested in mining RustChain (RTC)

**Prerequisites:** A computer with internet access, basic command line knowledge.

---

## 1. Introduction (0:00 - 0:30)

"Hello everyone! Welcome to this quick tutorial on installing and running the RustChain miner. RustChain is a Layer 1 blockchain built entirely in Rust, focused on speed and efficiency. By running a miner, you can earn RTC tokens and support the network. Let's get started!"

## 2. System Requirements (0:30 - 0:45)

"First, make sure you have:
- A modern CPU (multi-core recommended)
- At least 4GB of RAM
- Rust installed (if building from source) OR you can use the pre-built binaries.
- A RustChain wallet address."

## 3. Download the Miner (0:45 - 1:15)

"Head over to the official RustChain GitHub releases page: [link]. Download the appropriate binary for your operating system (Windows, macOS, Linux). Alternatively, you can clone the repository and build it yourself using `cargo build --release`."

## 4. Configure the Miner (1:15 - 1:45)

"Once downloaded, extract the archive. You'll find a `miner.toml` configuration file. Open it in a text editor. Set your wallet address in the `[wallet]` section, and optionally adjust the number of CPU threads in the `[mining]` section. Save the file."

## 5. Start Mining (1:45 - 2:30)

"Now open a terminal in the miner directory. On Linux/macOS, run `./rustchain-miner`. On Windows, run `rustchain-miner.exe`. You should see output showing accepted shares and your hash rate. Let the miner run for a few minutes to see it in action."

## 6. Verify Your Earnings (2:30 - 2:50)

"You can check your balance on the RustChain explorer: [link]. Enter your wallet address to see any mined RTC. Note that it may take some time for the first payout."

## 7. Conclusion (2:50 - 3:00)

"That's it! You are now mining RustChain. If you found this helpful, please like and subscribe. For more tutorials, check out the Elyan Labs documentation. Happy mining!"
