# RustChain 60-Second Explainer Video

This directory contains resources related to the 60-second technical explainer video for RustChain.

## Video Content Overview

The video covers these key concepts:
- What is RustChain and its core purpose
- Key architectural components
- How consensus works
- Benefits of using Rust for blockchain development
- Quick demonstration of basic operations

## Video Links

The completed 60-second explainer video can be found at:
