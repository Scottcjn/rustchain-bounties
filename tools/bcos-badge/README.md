# BCOS Badge Generator

A simple web tool to generate beautiful BCOS v2 badges for your projects.

## Features

- 🎨 Multiple badge styles (Flat, Flat Square, Plastic, For The Badge)
- 🏷️ Different status options (Validated, Pending, Failed, Scanning)
- 📋 One-click copy to clipboard
- 🎯 Live preview

## Usage

1. Open `index.html` in your browser
2. Enter your project name
3. Select status and style
4. Click "Generate Badge"
5. Copy the Markdown or HTML code

### GitHub Pages

1. Fork this repository
2. Go to Settings > Pages
3. Select main branch
4. Your badge generator will be live at `https://yourusername.github.io/bcos-badge-generator`
