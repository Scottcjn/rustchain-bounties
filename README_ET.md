## Mis on RustChain?

RustChain on blokiahelaplatvorm, mis on ehitatud Rust programmeerimiskeelele. See pakub:

- Turvalist ja kiiret tehingute töötlust
- Nutilepinguid Rust'is
- Skaleeritavat ja hajutatud arhitektuuri

## Paigaldus

```bash
# Kloonige hoidla
git clone https://github.com/rustchain/rustchain.git
cd rustchain

# Ehitage projekt
cargo build --release
```

## Käivitamine

```bash
# Käivitage sõlm
./target/release/rustchain-node --dev
```
