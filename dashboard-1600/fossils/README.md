## Overview

Interactive stratigraphy visualization of every RustChain attestation since genesis — like geological layers, but for silicon.

### Features

- **X-axis:** Time (epochs)
- **Y-axis:** Stacked by architecture family (oldest arch at bottom)
- **Color:** Architecture family
- **Width:** Number of attestations from that architecture in that epoch
- **Hover:** miner ID, device, RTC earned, fingerprint quality

## Usage

Serve the directory (e.g., with Python):

```bash
cd dashboard-1600/fossils
python3 -m http.server 8080
# Open http://localhost:8080
```

Or open `index.html` directly in a browser (requires a local server for data fetch):

```bash
cd dashboard-1600/fossils
python3 -m http.server 8080
```
