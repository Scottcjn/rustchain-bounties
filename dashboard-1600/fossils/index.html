The Fossil Record

This visualization renders every attestation from RustChain's genesis as geological strata — the older the architecture, the deeper the layer.

X-axis: Epoch (time). Y-axis: Stacked by architecture family. Width: Number of attestations from that architecture in that epoch. Color: Architecture family.

Hover over any stratum to see the miner's fingerprint data, epoch, and RTC earned.

Vertical dashed lines mark epoch boundaries. Vertical gold lines mark when a new architecture first appeared.
