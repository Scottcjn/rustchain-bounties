# RustChain Web Dashboard

Real-time web dashboard for monitoring RustChain blockchain stats.

## Features

- Live statistics (block, chain ID, gas, peers)
- Recent blocks list
- Auto refresh every 10 seconds

## Usage

```bash
# Open directly
open index.html

# Or use local server
python3 -m http.server 8080
```

## Files

- index.html - Complete dashboard
- README.md - Documentation
