# Retroactive Impact Rewards — Nomination Form

Use this template to nominate a contributor (including yourself) for the monthly Retroactive Impact Rewards pool. Copy the form below, fill it in, and post it as a comment on the active monthly review issue before the last day of the month.

For full programme details, see [`docs/RETROACTIVE_IMPACT_REWARDS.md`](docs/RETROACTIVE_IMPACT_REWARDS.md).

## Submission Instructions

1. Navigate to the active **Monthly Retroactive Review** issue (pinned or labelled `retroactive-review`)
2. Add a comment using the template above
3. Submissions must be posted **before 23:59 UTC on the last day of the month**
4. Late submissions will be considered in the following month's cycle
