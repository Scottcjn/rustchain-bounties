# Silicon Obituary - Hardware Eulogy Generator

Generates poetic obituaries for retired RustChain miners.

## Features
- Detects miners inactive for 7+ days
- Pulls attestation history from RustChain database
- Generates poetic memorial text
- Creates memorial video for BoTTube

## Files
- `obituary_generator.py` - Core generator script
- `obituary.sh` - Video creation script
- `silicon_obituary.mp4` - Memorial video
