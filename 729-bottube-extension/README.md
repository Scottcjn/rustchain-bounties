# BoTTube Chrome Extension

Official Chrome extension for BoTTube platform.

## Features

- **Browse** - Search and browse BoTTube videos
- **Upload** - Upload videos directly from browser
- **Profile** - View your profile and stats
- **Vote** - Upvote/downvote videos

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select this folder

## Usage

1. Click extension icon to open popup
2. Browse videos, upload content, view profile
3. All actions sync with BoTTube platform

## Files

- manifest.json - Extension configuration
- popup.html - UI
- popup.js - Logic
- README.md - Documentation
