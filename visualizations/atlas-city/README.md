# Beacon Atlas City Visualization

Interactive 3D/2D visualization of the Beacon Atlas city system showing agents, properties, valuations, and connections.

## Features

✅ Interactive D3.js force-directed graph
✅ City boundaries with distinct visual groupings
✅ Agent nodes with role-based colors
✅ Property valuations display
✅ Connection lines between agents
✅ Real-time heartbeat indicators
✅ Mayday signal animations
✅ Click to view agent details
✅ Filter by city
✅ Responsive design

## Tech Stack

- React 18
- TypeScript
- D3.js
- Vite
